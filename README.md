# file-organizer-challenge

Script to read input txt file and output a folder structure to the console.

## Built With
* [Node.js](https://nodejs.org/en/) - Node
* [TypeScript](https://www.typescriptlang.org/) - Typings

## Prerequisites
* Npm 
* Node.js
* TypeScript

## Getting Started
1. `npm i`
2. `npm run build`
3. `npm run start`
